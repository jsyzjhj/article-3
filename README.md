# 基于文件数据库的文章管理系统 
具有文件数据库的单入口文章管理系统，主要联系验证码类，同时
还有文章管理系统的各项功能。

## 总体思路 
通过使用不同的get参数inde.php入口文件实现文章的各个功能。
